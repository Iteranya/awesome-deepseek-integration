## Deepseek Integration

Adding Deepseek Integration Is Dead Easy

Just go to the AI Config Panel and do the following:

<img width="655" alt="AI Config Panel" src="https://github.com/user-attachments/assets/eed44b03-20ce-4c88-9e3d-e3f28d30afa2" />


```
Base LLM Model: deepseek-chat
AI Endpoint: https://api.deepseek.com/v1
Temperature: 1.3 (for roleplay)
AI API Key: sk-your-api-key
```

Just like that, you have finished integrating Deepseek with Viel!
